Para el funcionamiento de un robot de trading es necesario acceder a distintos tipos de datos. Estos datos garantizan la correcta toma de decisiones y la precisión en la ejecución del algoritmo.

## Tipos de datos:

1. **Datos de mercado:**
   - Precios actuales (cotizaciones) y volúmenes de negociación.
   - Libros de órdenes (profundidad de mercado) para el análisis de liquidez.

2. **Datos históricos:**
   - Se utilizan para probar estrategias (backtesting).
   - Incluyen precios, volúmenes y eventos del mercado de períodos anteriores.

3. **Datos fundamentales:**
   - Informes financieros de empresas, noticias, estadísticas macroeconómicas.
   - Importantes para estrategias a largo plazo.

4. **Datos de eventos:**
   - Información sobre eventos corporativos como dividendos, fusiones y adquisiciones.

## ¿Dónde obtener datos?

- **[AlphaVantage](https://alphavantage.co/):** Datos históricos para diversas bolsas.
- **[Yahoo Finance](https://finance.yahoo.com/):** Datos gratuitos para el análisis de acciones e índices.
- **[Quandl](https://www.quandl.com/):** Datos fundamentales y de mercado para análisis.
- **[Interactive Brokers](https://www.interactivebrokers.com/):** API para acceso a datos de mercado en tiempo real.

## Consejos:

- Verifique la calidad de los datos antes de utilizarlos.
- Para el trading de alta frecuencia, elija fuentes con latencia mínima.
- Archive los datos para análisis y comparación de resultados.
