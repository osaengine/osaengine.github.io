StockSharp has published an announcement that an extensive collection of trading strategies in C# and Python is now available on GitHub, ready to use in Designer, Runner, Shell, or via API.

The collection features hundreds of open-source algorithmic strategies for various trading platforms. All strategies are available for download, study, and independent deployment. Detailed instructions allow you to quickly set up the strategies locally or integrate them into your own projects.

The repositories include documentation for each solution, describing parameters and configuration options. Users are free to modify the code for their own needs or use it for learning and testing.
