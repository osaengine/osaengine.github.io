## 什么是 MCP

**Model Context Protocol (MCP)** 是由 [Anthropic](https://www.anthropic.com/) 开发的开放协议，用于标准化 AI 模型连接外部数据源和工具的方式。简单来说：MCP 就是 AI 的"USB 接口"。

在 MCP 出现之前，每次 AI 与外部服务的集成都需要编写定制代码。MCP 提供了一个统一标准，基于**客户端-服务器**模型运作：

```
AI 模型（客户端）<-> MCP 协议 <-> MCP 服务器 <-> 外部服务
```

## MCP 如何与券商账户协作

MCP 架构提供三种交互类型：

### 1. Resources（资源）
获取数据：报价、投资组合、交易历史、账户余额。

### 2. Tools（工具）
执行操作：下单、修改订单、取消持仓。

### 3. Prompts（提示）
预定义的交互模板："显示我的投资组合"、"分析我本周的交易历史"。

## 交互示例

想象一下与 AI 助手的对话：

> **你**：我的投资组合中目前有哪些持仓？
>
> **AI**（通过 MCP 查询券商）：您的投资组合有 3 个持仓：
> - AAPL：50 股，+4.2%
> - GOOGL：20 股，-1.1%
> - SBER：100 股，+2.8%
>
> 投资组合总收益率：+3.1%

甚至可以：

> **你**：在 $210 的价位给 AAPL 设置止损。
>
> **AI**：已创建以 $210 卖出 50 股 AAPL 的止损单。请确认？

## 安全性

安全性是金融集成的核心问题。MCP 提供了：

- **OAuth 2.0 授权** — 安全访问的标准协议
- **细粒度权限** — 可以只允许数据读取而不允许交易
- **审计日志** — 所有操作都会被记录
- **操作确认** — 关键操作需要用户明确同意

## 谁已经支持 MCP

截至 2026 年 3 月，以下服务提供了 MCP 服务器：

- **TradeStation** — 首个提供完整 MCP 集成的券商
- **FactSet**、**S&P Global**、**MSCI** — 通过 Anthropic 的金融插件
- 其他券商和平台正在探索集成方案

## 这对算法交易意味着什么

MCP 打开了通向 **AI 管理交易**的大门，模型不仅能生成信号，还能：

1. 自主分析市场状况
2. 形成交易决策
3. 通过券商执行交易
4. 监控结果并调整策略

这并不意味着应该让 AI 完全掌控你的账户。但**人在回路中**的方式——AI 提出建议，人来确认——已经成为现实。

## 如何开始

如果你想体验 MCP：

1. 安装 [Claude Desktop](https://claude.ai/download) 或其他 MCP 兼容客户端
2. 连接你的券商 MCP 服务器
3. 从只读模式开始——只查看数据
4. 随着信任度的提高，逐步添加更多功能

MCP 不是一夜之间的革命，而是人类与 AI 在金融领域交互新时代的基石。
