StockSharp hat auf seiner Website angekündigt, dass auf GitHub eine umfangreiche Sammlung von Handelsstrategien in C# und Python verfügbar ist, die sofort in Designer, Runner, Shell oder über die API eingesetzt werden können.

Die Sammlung umfasst Hunderte von algorithmischen Open-Source-Strategien für verschiedene Handelsplattformen. Alle Strategien stehen zum Download, zum Studium und zur eigenständigen Nutzung bereit. Detaillierte Anleitungen ermöglichen es, die Strategien schnell lokal einzurichten oder in eigene Projekte zu integrieren.

Die Repositories enthalten Dokumentation für jede Lösung mit Beschreibungen der Parameter und Konfigurationsmöglichkeiten. Benutzer können den Code frei für ihre eigenen Bedürfnisse anpassen oder ihn zum Lernen und Testen verwenden.
