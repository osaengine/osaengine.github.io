StockSharp ha publicado un anuncio sobre la disponibilidad en GitHub de una extensa colección de estrategias de trading en C# y Python, listas para usar en Designer, Runner, Shell o mediante API.

La colección incluye cientos de estrategias algorítmicas de código abierto para diversas plataformas de trading. Todas las estrategias están disponibles para descargar, estudiar y desplegar de forma independiente. Las instrucciones detalladas permiten configurar rápidamente las estrategias de forma local o integrarlas en sus propios proyectos.

Los repositorios incluyen documentación para cada solución, describiendo los parámetros y opciones de configuración. Los usuarios pueden modificar libremente el código para sus propias necesidades o utilizarlo para aprendizaje y pruebas.

Más detalles sobre el proyecto y enlaces a la colección están disponibles en el sitio web de StockSharp:
