## Что такое MCP

**Model Context Protocol (MCP)** — это открытый протокол, разработанный [Anthropic](https://www.anthropic.com/), который стандартизирует способ подключения ИИ-моделей к внешним источникам данных и инструментам. Если упростить: MCP — это «USB-порт» для ИИ.

До появления MCP каждая интеграция ИИ с внешним сервисом требовала написания кастомного кода. MCP предлагает единый стандарт, который работает по модели **клиент-сервер**:

```
ИИ-модель (клиент) <-> MCP-протокол <-> MCP-сервер <-> Внешний сервис
```

## Как MCP работает с брокерскими счетами

Архитектура MCP предусматривает три типа взаимодействия:

### 1. Resources (Ресурсы)
Получение данных: котировки, портфель, история сделок, баланс счёта.

### 2. Tools (Инструменты)
Выполнение действий: размещение ордеров, модификация заявок, отмена позиций.

### 3. Prompts (Промпты)
Предопределённые шаблоны взаимодействия: «Покажи мой портфель», «Проанализируй историю сделок за неделю».

## Пример взаимодействия

Представьте диалог с ИИ-ассистентом:

> **Вы**: Какие позиции сейчас в моём портфеле?
>
> **ИИ** *(через MCP обращается к брокеру)*: В вашем портфеле 3 позиции:
> - AAPL: 50 акций, +4.2%
> - GOOGL: 20 акций, -1.1%
> - SBER: 100 акций, +2.8%
>
> Общая доходность портфеля: +3.1%

Или даже:

> **Вы**: Поставь стоп-лосс на AAPL на уровне $210.
>
> **ИИ**: Создал стоп-ордер на продажу 50 акций AAPL по цене $210. Подтвердите?

## Безопасность

Вопрос безопасности — ключевой для финансовых интеграций. MCP предусматривает:

- **OAuth 2.0 авторизацию** — стандартный протокол для безопасного доступа
- **Гранулярные права доступа** — можно разрешить только чтение данных без торговли
- **Аудит-логи** — все действия записываются
- **Подтверждение операций** — критические действия требуют явного согласия пользователя

## Кто уже поддерживает MCP

На март 2026 года MCP-серверы доступны для:

- **TradeStation** — первый брокер с полноценной MCP-интеграцией
- **FactSet**, **S&P Global**, **MSCI** — через финансовые плагины Anthropic
- Другие брокеры и платформы изучают возможности интеграции

## Что это значит для алготрейдинга

MCP открывает путь к **ИИ-управляемой торговле**, где модель не просто генерирует сигналы, а может:

1. Самостоятельно анализировать рыночную ситуацию
2. Формировать торговые решения
3. Исполнять их через брокера
4. Мониторить результаты и корректировать стратегию

Это не означает, что нужно давать ИИ полный контроль над счётом. Но возможность **человек-в-цикле** — когда ИИ предлагает, а человек подтверждает — уже здесь.

## Как начать

Если вы хотите поэкспериментировать с MCP:

1. Установите [Claude Desktop](https://claude.ai/download) или другой MCP-совместимый клиент
2. Подключите MCP-сервер вашего брокера
3. Начните с read-only режима — только просмотр данных
4. Постепенно добавляйте возможности по мере роста доверия

MCP — это не революция одного дня, а фундамент для новой эры взаимодействия человека и ИИ в финансах.
