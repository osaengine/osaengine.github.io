## Миллион токенов: зачем это нужно

Google продолжает делать ставку на длинный контекст. **Gemini 3.1 Pro** сохраняет рекордное окно в **1 миллион токенов** — это примерно 700 000 слов, или несколько полноценных книг.

Для практических задач это означает:

- Загрузка **целой кодовой базы** среднего проекта в один запрос
- Анализ **годовых финансовых отчётов** без потери контекста
- Работа с **длинными историями переписок** и документации
- Обработка **многочасовых транскрипций** встреч и звонков

## ARC-AGI-2: тест на абстрактное мышление

Бенчмарк [ARC-AGI-2](https://arcprize.org/) проверяет способность модели к абстрактному рассуждению — задачи, которые легко решает ребёнок, но которые ставят в тупик большинство ИИ-систем.

Gemini 3.1 Pro показала **77% на ARC-AGI-2**, что является одним из лучших результатов среди коммерческих моделей:

| Модель | ARC-AGI-2 |
|--------|-----------|
| Claude Sonnet 4.6 | 79.2% |
| **Gemini 3.1 Pro** | **77.0%** |
| GPT-5.3 | 74.5% |
| DeepSeek V3 | 71.3% |

## Рыночная доля растёт

По данным аналитиков, доля Google на рынке LLM API выросла с 12% до 18% за последние шесть месяцев. Основные причины:

### Ценовая политика

Google предлагает одни из самых конкурентных цен:

- **Input**: $3.50 / 1M токенов
- **Output**: $10.50 / 1M токенов
- Для контекста >128K: $7 / $21 за миллион

### Экосистема Google Cloud

Интеграция с **Vertex AI**, **BigQuery** и другими сервисами Google Cloud делает Gemini привлекательным для корпоративных клиентов, которые уже используют облачную инфраструктуру Google.

### Мультимодальность

Gemini 3.1 Pro нативно поддерживает:

- Текст, изображения, аудио и видео
- Генерацию и анализ кода
- Работу с таблицами и структурированными данными

## Что это значит для трейдеров

Длинный контекст в 1M токенов открывает интересные возможности:

1. **Загрузка всей истории торгов** за длительный период для анализа паттернов
2. **Одновременный анализ** множества финансовых отчётов компаний
3. **Обработка новостного потока** за целый день без потери важных деталей

При этом нужно учитывать, что качество работы с информацией в начале и конце длинного контекста может отличаться — так называемая проблема «lost in the middle».

Google уверенно закрепляется в тройке лидеров рынка ИИ, и Gemini 3.1 Pro — серьёзный аргумент в этой гонке.
